# プライバシーポリシー

**最終更新日: 2025年12月23日**

NowOnMeet（以下「当方」といいます）は、スマートフォン用アプリケーション「ギフト手帳」（以下「本アプリ」といいます）におけるユーザー情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

## 1. 収集する情報とその利用目的

### 1.1 連絡先情報（Contacts）
本アプリは、ユーザーの端末に保存されている連絡先情報へのアクセス権限を要求します。
- **利用目的:** ギフトの贈り主や送り先の情報を入力する際、端末の連絡先から名前や関連情報を引用入力するために使用します。
- **データの取り扱い:** 取得した連絡先情報は、アプリ内のローカルデータベースにのみ保存され、入力補助の目的以外で使用されることはありません。**当方のサーバーへ送信されることは一切ありません。**

### 1.2 アプリ内データ（Gift Note Data）
ユーザーが本アプリに入力したギフトの記録、メモ、その他のデータは、ユーザーの端末内のローカルストレージ（データベース）に保存されます。
- **データの取り扱い:** これらのデータは端末内でのみ管理され、外部へ送信されることはありません。ユーザー自身がバックアップ操作を行わない限り、アプリを削除するとデータも削除されます。

### 1.3 利用状況データの取得（将来的な導入の可能性を含む）
本アプリでは、アプリの品質向上や不具合の改善を目的として、Google Firebase Analytics等の解析ツールを利用する場合があります。これにより、個人を特定しない形式で、アプリの利用頻度やクラッシュ情報などの統計データが収集される可能性があります。

## 2. 広告・アフィリエイトについて

### 2.1 Amazonアソシエイト・プログラム
本アプリは、Amazon.co.jpを宣伝しリンクすることによってサイトが紹介料を獲得できる手段を提供することを目的に設定されたアフィリエイトプログラムである、Amazonアソシエイト・プログラムの参加者です。
本アプリ内のリンクを経由してAmazon.co.jpで商品を購入された場合、当方は紹介料を獲得することがあります。

## 3. 第三者への提供

当方は、法令に基づく場合を除き、ユーザーの同意なく個人情報を第三者に提供することはありません。

## 4. プライバシーポリシーの変更

当方は、必要と判断した場合には、ユーザーに通知することなくいつでも本ポリシーを変更することができるものとします。
変更後のプライバシーポリシーは、本ページに掲載したときから効力を生じるものとします。ユーザーは、定期的に本ポリシーを確認することをお勧めします。
